# Orbit Breaker VR

A small WebXR arcade target game. Aim at orbiting cores and fire to score before the timer reaches zero.

## Run

From this folder:

```powershell
& "C:\Users\great\.cache\codex-runtimes\codex-primary-runtime\dependencies\node\bin\node.exe" "C:\Users\great\Documents\Codex\2026-06-16\a-vr-game-with-webxr\work\static-server.mjs"
```

Then open:

```text
http://127.0.0.1:8123/
```

Use a WebXR-capable browser and headset for the VR mode. The screen mode works with a mouse or touch.
